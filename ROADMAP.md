# Roadmap

## v0.1

- Buildable template package with main and item templates.
- Core, pro, and advanced profile smoke coverage.
- Core dispatcher default and optional MediatR path.
- TaskHub reference sample.
- Spec-driven development layer and AI guardrails.
- GitHub Actions for guardrails and template smoke tests.

## v0.2

- Expand generated integration tests with live PostgreSQL/Testcontainers.
- Add deeper contract tests for module public contracts.
- Add richer outbox dispatcher implementation.
- Add published package release workflow after maintainer approval.

## v1.0

- Stable public package release.
- Versioned template compatibility policy.
- Full docs site and migration guide.
