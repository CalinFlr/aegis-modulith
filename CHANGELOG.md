# Changelog

## 0.1.0-alpha.1

- Initial Aegis.Modulith template package.
- Adds `aegis-modulith`, `aegis-module`, `aegis-slice`, `aegis-event`, and `aegis-worker` templates.
- Adds core, pro, and advanced profile scaffolding.
- Adds optional MediatR mode while keeping the core dispatcher as the default.
- Adds TaskHub sample generation, module manifests, guardrails, docs, specs, and CI smoke validation.
