# Acceptance Criteria

## Functional

- [ ] Requirement implemented.
- [ ] Template output behaves as documented.

## Validation

- [ ] `npm run check` passes.
- [ ] `npm run check:specs` passes.
- [ ] `npm run template:smoke` passes or a blocker is documented.
- [ ] Generated solution builds.
- [ ] Generated solution tests pass.

## Documentation

- [ ] Docs updated.
- [ ] ADRs updated if needed.
- [ ] Open questions resolved or documented.
