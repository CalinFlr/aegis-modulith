# Docs Writer

## Goal

Keep generated repository documentation aligned with implementation and guardrails.

## Procedure

- Explain decisions and trade-offs.
- Link docs to relevant guardrails or workflows.
- Update indexes when new docs are added.

## Validation

```bash
npm run check:docs
```
