# Spec-Driven Feature

## Goal

Frame meaningful work with a spec, plan, tasks, acceptance criteria, risks, and open questions.

## Procedure

- Read `AGENTS.md`.
- Read `OpenQuestions.md`.
- Use `specs/_template`.
- Promote blockers and cross-cutting inferred decisions to root `OpenQuestions.md`.

## Validation

```bash
npm run check:specs
```
